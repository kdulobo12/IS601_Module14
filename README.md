# 🧮 Module 14 – BREAD Calculations Application

## 📌 Project Description

This project is a full-stack FastAPI application that implements BREAD (Browse, Read, Edit, Add, Delete) functionality for user-specific calculations.

Each authenticated user can:
- Add new calculations (add, subtract, multiply, divide)
- View all their calculations
- View a specific calculation
- Edit existing calculations
- Delete calculations

The project also includes:
- PostgreSQL database integration
- JWT authentication
- Pytest backend testing
- Playwright E2E testing
- Docker containerization
- GitHub Actions CI/CD pipeline
- Docker Hub deployment
